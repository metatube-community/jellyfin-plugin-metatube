## 关于

为 Jellyfin/Emby 开发的超级好用的元数据插件。

## 特性

- 完整数据：包括标题、**简介**、演员、标签、**评分**等内容。
- 完整搜索：支持通过众多的刮削源搜索影片和演员信息。
- 预告功能：无需下载完整预告视频即可**在线观看预告片**。
- 计划任务：自动整理影片标签以及在后台自动更新插件。
- 人脸识别：内置的人脸识别以人脸为中心裁剪海报图像。
- 自动翻译：支持将特定的元数据内容翻译成需要的语言。
